Write you logic in Javascript and your UI in QML

Keep your logic and UI cleanly separated. No need to write a C++ backend for your app either.

Use classes, decorators, arrow functions, and more

Quickly uses QMLify and Babel to transpile modern ES6 into Javascript that is compatible with QML. You can also just use the Quickly library in existing projects without the QMLify transpiler.

{% highlight js %} import {field, register, Document} from 'sphere' connect('quickly-demo', 'Sample demo of Quickly') export class Person extends Document { @field('string') firstName @field('string') lastName @field('string') email @field('date') createdDate @field('bool') notificationsEnabled constructor(firstName, lastName, email) { this.firstName = firstName this.lastName = lastName this.email = email } } register(Person) {% endhighlight %}

Naturally use your code from QML

The Quickly library and your own QMLified code can be used naturally from QML using standard imports, just as you would with QML singletons or standard QML-style JS libraries. No need for using require(), as other Node.js/CommonJS integrations for QML do. You can also publish your code for other developers to use without them having to use QMLify or Quickly.

Read from and write to the filesystem without C++

You can use the fs module, as well as path, url, assert, and other core Node.js modules. In addition, many ES6 polyfills are provided.

{% highlight js %} import fs from 'fs' if (!fs.exists('config.json')) throw new Error('File doesn\'t exist!') const config = JSON.parse(fs.readFile('config.json')) fs.writeFile('output.log', 'Configuration loaded.') {% endhighlight %}

Use fetch and promises for network access

No need to use XMLHttpRequest. Quickly provides a polyfill for the fetch API and ES6 Promises, which makes network access a breeze.

Use many of the 250,000+ npm packages

Many of the packages available via npm work with Quickly. And if they don't, submit a bug and we can ship a patch with Quickly so the module works for you and everyone else.

{% highlight js %} import _ from 'lodash' _.assign({ 'a': 1 }, { 'b': 2 }, { 'c': 3 }); // → { 'a': 1, 'b': 2, 'c': 3 } _.map([1, 2, 3], function(n) { return n * 3; }); // → [3, 6, 9] {% endhighlight %}
